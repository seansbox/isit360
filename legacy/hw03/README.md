# HW03 Structuring Data and Querying It

## Installing The Tools

### Mac

    brew install sqlite3
>
    brew install --cask sqlitestudio
>
    pip install sqlite-web


### Windows

    choco install sqlite
>
    choco install sqlite-studio.portable
>
    pip install sqlite-web
